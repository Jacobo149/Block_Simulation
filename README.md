# Block_Simulation

Learning tool with a command line interface.  
Simulates Blockchain Block creation, walking users through the steps along the way.


## How to Run

### Prerequisites
- Download a C compiler
- Clone Repository
- Navigate to Repository

### Console Commands
Compile Project
```
make
```
Navigate to bin directory
```
cd bin
```
Run executable
```
.\main
```
